# EDI Rent A Car

Kullanıcıların kolayca araç kiralayabileceği, dinamik ve modern bir web tabanlı araç kiralama platformu. Bu proje, kullanıcı deneyimini ön planda tutarak geliştirilmiş olup, arka planda operasyonel süreçlerin yönetilebilmesi için bir admin paneli altyapısı barındırmaktadır.

## Özellikler

- **Kullanıcı Dostu Arayüz:** CSS ile tasarlanmış modern, temiz ve duyarlı (responsive) web tasarımı.
- **Araç Listeleme:** Mevcut araçların detaylı bilgileriyle (model, yıl, fiyat) görüntülenmesi.
- **Kiralama İşlemleri:** Kullanıcıların araç seçimi ve rezervasyon adımlarını kolayca tamamlayabilmesi.
- **Admin Paneli:** Yönetici tarafında araç ekleme, çıkarma ve kiralama durumlarının takibi için tasarlanmış yönetim modülü.
- **Veritabanı Mimarisi:** Verilerin güvenli ve düzenli bir şekilde tutulmasını sağlayan veritabanı tasarımı.

## Kullanılan Teknolojiler

- **Frontend:** HTML5, CSS3, JavaScript
- **Backend / Veritabanı:** MySQL 
- **Tasarım:** Modern UI/UX prensipleri
